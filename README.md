# retail-sales-analytics-project
This project analyzes 100,000 retail sales transactions using Python, SQL, and Power BI. The objective was to identify sales trends, customer purchasing behavior, product performance, and discount effectiveness to support data-driven business decisions. 

Key analyses include:
Revenue by product category
Revenue by sales region
Monthly sales trends
Average discount by product category
Revenue by customer age group
Revenue by discount level

Tools Used:
Python (Pandas, Matplotlib)
Jupyter Notebook
SQL (SQLite)
Power BI

Dataset Notes:
The dataset contains sales records from January through September. October, November, and December were not included in the source data, so monthly trend analysis is limited to the available time period.
